## Analytics Projects (Massey University — Semester 1, 2026)

### 1. 🐍 Credit Risk Analysis — Python & SQL
**Course:** 161.777 Practical Data Mining
**Files:** `Assessment_4_Credit_Risk_Analysis.ipynb` | `Assessment_3_SQLite_Database.ipynb`

End-to-end credit risk project integrating **three independent data sources**:
- **Kaggle** — static borrower-level credit risk dataset
- **World Bank API** — macroeconomic indicators (inflation, unemployment, lending rates, 2018–2024)
- **RBNZ website** — web-scraped Official Cash Rate (OCR) data

**Key findings:**
- Loan grade is the strongest single predictor of default — Grade G borrowers default at ~4× the rate of Grade A
- Renters show consistently higher default rates than mortgage holders
- Higher interest rates concentrate in lower-grade loans, compounding risk exposure

**SQLite database** — normalised 6-table schema (3NF) with views, indexes, CHECK constraints, and join-based analytical queries

**Skills demonstrated:** Python, pandas, requests, BeautifulSoup, matplotlib, API integration, web scraping, SQL, database design, normalisation, EDA, credit risk analysis
